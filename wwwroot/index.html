¿Por qué base documental?

MongoDB ofrece un equilibrio entre estructura y capacidad de consulta. A diferencia de modelos como clave-valor, grafos o columnares que están super especializados en algo particular, las bases de datos documentales permiten flexibilidad tanto en la inserción como en la consulta de los datos sin sacrificar lo estructural del sistema.

// Pedidos por cliente
db.pedidos.aggregate([
  { $group: { _id: "$cliente.nombre", cantidadPedidos: { $sum: 1 }, totalGastado: { $sum: "$total" } } },
  { $sort: { cantidadPedidos: -1 } }
])
// Top clientes
db.pedidos.aggregate([
  { $group: { _id: "$cliente.nombre", cantidadPedidos: { $sum: 1 }, totalGastado: { $sum: "$total" } } },
  { $sort: { cantidadPedidos: -1, totalGastado: -1 } },
  { $limit: 5 }
])
// Productos más vendidos
db.pedidos.aggregate([
  { $unwind: "$productos" },
  { $group: { _id: "$productos.nombre", cantidadVendida: { $sum: "$productos.cantidad" } } },
  { $sort: { cantidadVendida: -1 } }
])
// Ventas por día
db.pedidos.aggregate([
  { $group: { _id: { $dateToString: { format: "%Y-%m-%d", date: "$fecha" } }, totalVentas: { $sum: "$total" }, cantidadPedidos: { $sum: 1 } } },
  { $sort: { _id: 1 } }
])
// Pedidos por estado
db.pedidos.aggregate([
  { $group: { _id: "$estado", cantidad: { $sum: 1 } } },
  { $sort: { cantidad: -1 } }
])
